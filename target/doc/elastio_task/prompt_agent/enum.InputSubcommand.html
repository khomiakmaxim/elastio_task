pub enum InputSubcommand {
    Configure(ProviderName),
    Get(SpaceTimeConfig),
    CurrentProvider,
}
Expand description

Enumeration for possible input subcommand variants.

Variants§

§

Configure(ProviderName)

Configures provider for retrieving weather data. Example: configure weather-api (open-weather-map is default)

§

Get(SpaceTimeConfig)

Gets apporpriate weather data, based on address and date(YYYY-MM-DD), if provided, and current weather, if not. Example: get “L’aquila, Italy” 2023-04-07

§

CurrentProvider

Displays currently used provider.
